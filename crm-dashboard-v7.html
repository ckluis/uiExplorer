CRM Dashboard

v7 Fully Autonomous

Purpose

A relationship graph CRM for real estate agents that visualizes contacts as nodes and connections as edges. Reveals network structure, influence clusters, and referral paths through a constellation-style force-directed graph rendered entirely in CSS.

Variant Focus

"Web" — See who knows who. Contacts are nodes sized by value, relationships are edges styled by type, and proximity reveals influence clusters. Dark, contemplative, like a planetarium.

Prompt

This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

v7: "Web" — Relationship Graph

Tagline: "See who knows who."
Core metaphor: A constellation map or neural network — contacts are nodes, relationships are edges, proximity reveals influence.

Layout: Central area (~75%) is a force-directed-style graph in pure CSS — absolutely positioned circular nodes with CSS lines/borders connecting them. Left panel (~25%, ~320px) with detail card for selected node, search, and filters. Dark background to make graph elements pop.

Key UI elements: Contact nodes sized by value (40-80px) with initials/gradient avatar. Relationship edges (solid=direct client, dashed=referral, dotted=weak). Cluster halos grouping related nodes. Node hover detail with glow. Detail panel with connections list. Orphan node indicators. Referral path highlighting. Time decay (fading edges).

Color/mood: Dark background (#1a1a2e). Nodes in pastel-to-saturated palette. Edges in semi-transparent white. Low-opacity cluster halos. Contemplative, discovery-oriented, like a planetarium.

Role toggle: Solo Agent shows personal network. Team Lead merges team networks with shared connection highlights. Brokerage Admin shows full brokerage network with agent hub nodes and top connectors ranking.
