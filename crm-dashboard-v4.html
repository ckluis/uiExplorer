CRM Dashboard

v4 Fully Autonomous

Purpose

A geography-first CRM for real estate agents. The map is the primary organizing principle — contacts, listings, and deals appear as positioned pins on an abstract neighborhood grid. Left panel provides list, detail, and stats views. Role toggle shifts between solo agent territory, team coverage, and brokerage-wide view.

Variant Focus

"Territory" — Your market, visualized. Military campaign map metaphor where geography drives all CRM interactions. Map-centric with ~70% viewport for spatial view.

Prompt

This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

v4: "Territory" — Map-Centric CRM

Tagline: "Your market, visualized."
Core metaphor: A military campaign map — geography is the primary organizing principle with contacts, listings, and deals as pins.

Layout: The "map" occupies ~70% of viewport (right side) — rendered as a stylized CSS grid/abstract map with positioned elements representing a neighborhood layout. Left panel (~30%, ~400px) as contextual list/detail view. Top bar with role toggle, search, and layer toggles.

Key UI elements: Color-coded map pins (blue circles for buyer leads, green house icons for listings, amber diamonds for deals). Cluster indicators. Heat zone overlays. Left panel with tabs: List, Detail, Stats. "Farm area" highlighted zone. Property card overlays. Layer toggle checkboxes. Abstract street grid in CSS.

Color/mood: Muted map base (light gray streets, subtle terrain). Saturated pins and overlays. Clean white left panel. Professional cartography meets data visualization. Cool-toned.

Role toggle: Solo Agent shows personal leads/listings/deals with personal territory. Team Lead shows all team pins with agent color rings and coverage gaps. Brokerage Admin shows entire territory with agent boundary overlays and market share.
