Purpose

Enhanced iteration of the Kitchen Brigade (v6) — adds device preview framework and CSS-only interactions. Part of a 20-variant research series.

Variant Focus

Device preview + checkbox station tickets, hover task/urgency, radio buttons for service time phases (prep/service/cleanup).

Prompt

This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

Build agent-orchestration-v16.html as an enhanced iteration of v6 (Kitchen Brigade). Take the existing Kitchen Brigade floor plan and make two changes: 1. Wrap in device preview window framework (desktop/mobile/fullscreen). 2. Add CSS-only interactions: checkbox toggles per station to show/hide detailed ticket breakdown, :hover on chef/commis reveals current task and urgency, radio buttons for service time phases (prep/service/cleanup). Mobile: list view instead of floor plan, collapsible station details. Preserve warm kitchen palette.
