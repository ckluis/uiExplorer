v3: "Mobile Fixer" — Bottom Sheet with One-Tap Fixes

Tagline: "Fix it now, you're in the field."

Core Concept: A mobile-first bottom sheet interface optimized for field workers who need to fix issues while on-the-go. Large touch targets, thumb-zone layout, one-tap fixes. No complex forms — just quick A/B choices.

Exposure Method: Bottom tab bar icon with red warning badge showing count. Tapping reveals bottom sheet.

Presentation Style: iOS-style bottom sheet that swipes up from bottom. Drag handle at top. Issues presented as large cards with one-tap actions.

Target Persona: Mobile casual user, field worker (sales rep, technician, etc.). Needs to fix things quickly between appointments. Often outdoors, possibly wearing gloves.

Platform: Mobile-first (375x720 primary view). Desktop shows centered mobile frame.

Key UI Elements: Bottom tab bar with Issues icon + badge, Drag handle on bottom sheet, Large touch targets (48px minimum), Thumb-zone layout (actions near bottom), Simple A/B choice buttons ("Option A" | "Option B"), Haptic feedback pulse indicators (visual ripples), Quick fix confirmations (green checkmark toast), Swipe-to-dismiss on resolved items.

Sample Checklist Items: 1. "Customer address outdated" — Update now | Ask customer. 2. "Appointment overlap detected" — Keep earlier | Keep later. 3. "Missing product photo" — Take photo | Skip. 4. "Route needs optimization" — Optimize | Keep current. 5. "Check-in required" — Check in now | Remind later.

Color/Mood: Clean, high-contrast for outdoor visibility. iOS-inspired with system blue accents. Large, readable typography.

Interaction Patterns: Checkbox toggle for bottom sheet visibility, :checked state shows sheet at different heights (peek/full), Large button tap states, Swipe gesture indicators.

Device Preview: Include switcher, but default to mobile view. Desktop shows phone frame.
