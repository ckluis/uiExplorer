CRM Dashboard

v6 Fully Autonomous

Purpose

A terminal-inspired, command-line CRM for power-user real estate agents. Dense monospace data tables, keyboard shortcut legend, command bar with autocomplete, and tmux-style split panels. Optimized for agents who think faster than they click.

Variant Focus

"Terminal" — Command-Line CRM. Bloomberg Terminal meets real estate: dense text, monospace everything, blinking cursors, color-coded status indicators, and a command input bar for rapid navigation.

Prompt

This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

v6: "Terminal" — Command-Line CRM

Tagline: "For agents who think faster than they click."
Core metaphor: A developer's terminal or Bloomberg Terminal — dense, keyboard-driven, text-based, brutally fast.

Layout: Dark background, full screen. Persistent command input bar at bottom (~48px, monospace, blinking cursor via CSS animation). Main area divided into panels (tmux-style): wide main panel with dense tabular results, narrower side panel with watchlist/pinned contacts. Status bar at top with key metrics.

Key UI elements: Command input bar with autocomplete suggestions. Dense monospace table with alternating rows and color-coded cells. Status bar with blinking indicators. Record detail view in key:value pairs. Shortcut legend panel. Watchlist panel. Notification ticker with CSS animation. Split-view panels.

Color/mood: Dark mode exclusively. Background: #0d1117. Text: #c9d1d9. Terminal palette: green (#3fb950), yellow (#d29922), red (#f85149), cyan (#58a6ff). JetBrains Mono font. Powerful, focused, no-nonsense.

Role toggle: Solo Agent queries personal data. Team Lead adds team metrics and agent leaderboard. Brokerage Admin shows brokerage-wide metrics and agent comparison panels.
