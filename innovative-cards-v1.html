Innovative Cards

v1

Purpose

Explores 7 different card archetypes — e-commerce, SaaS pricing, recipe, map/directions, profile, music, and news — each with 3-4 CSS-only iterations showing different interaction patterns: hover reveals, checkbox flips, radio toggles, animated waveforms, and more.

Prompt

It's time for a new experiment. Now I want to play with really innovative new cards. There should be a collection of cards that do different things and have different interactions. Since some of them should be visual - feel free to use a public website that hosts images for embeds and allows it. There should be ecommerce, saas, recipe, directions, maps, you name it. Go WILD.

Importantly there should be 3-4 iterations for each card in this one page. Future iterations may focus on one type, but we need to know what works and what doesn't. Also experiment with interaction patterns (click, hover, flip, etc).
