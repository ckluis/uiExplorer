Build guided-checklists-v8.html — the "Proactive Monitor" variant for the Guided Checklists experiment series.

Name: Proactive Monitor
Tagline: "This will become a problem in 3 days."
Version: v8

Core Concept: A notification drawer focused on FUTURE problems, not just current ones. Shows predictive warnings with countdown timers. Emphasizes early intervention before issues become critical.

Exposure Method: Menu bar notification bell with severity-colored badge. Badge shows count and most severe level (red/amber/green dot).

Presentation Style: Notification drawer (~360px) sliding from right. Grouped by severity with time predictions.

Target Persona: Operations admin, DevOps-minded user. Values early warning, preventive maintenance. Proactive rather than reactive.

Platform: Web (desktop), with mobile drawer adaptation.

Key UI Elements:
- Notification bell icon with colored badge
- Severity grouping (Critical predictions, Warnings, Monitoring)
- Time-based predictions ("Will become critical in 3 days")
- Countdown timers (visual progress bars depleting)
- Snooze options with duration picker
- Acknowledge button (removes from active)
- Escalation paths ("Escalate to: Team Lead")
- Trend arrows (getting worse, improving)
- "Dismiss as false positive" option (trains model)

Sample Predictions:
1. CRITICAL IN 3 DAYS: "Storage will exceed quota" — Currently 87%, growing 5%/day
2. WARNING IN 1 WEEK: "SSL certificate expires" — Expires Feb 13
3. WARNING IN 2 DAYS: "API rate limit approaching" — Currently 78% of limit
4. MONITORING: "Memory usage trending up" — 62%, +2% this week

Color/Mood: Alert-focused but not alarming. Uses countdown aesthetic. Red/amber/green severity system. Dark drawer with light content.

AI Feedback Pattern:
- "Dismiss as false positive" trains prediction model
- Snooze duration signals urgency perception
- Acknowledge without fixing = accepting prediction
- Escalation = agreeing with severity assessment

Interaction Patterns:
- Checkbox toggles drawer open/closed
- Radio buttons for severity filter (All/Critical/Warning/Info)
- CSS animations for countdown progress bars
- Hover on predictions shows detailed reasoning

Device Preview: Include desktop/mobile/fullscreen switcher.
