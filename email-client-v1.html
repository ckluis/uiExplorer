Email Client

v1 Fully Autonomous

Purpose

Explores a classic 3-pane email client layout (folder list, message list, reading pane) using only CSS radio buttons for state management. Tests whether a rich, interactive-feeling mail UI can be built with zero JavaScript.

Prompt

This version was generated autonomously. The prompt shown is a summary of the agent's intent, not a verbatim human prompt.

v1: Basic 3-pane layout with folders (Inbox, Sent, Drafts, Trash, Archive) using radio buttons for folder selection, a message list with sender/subject/preview/timestamp, and a reading pane that shows the selected message content. Includes unread message styling with a left border accent.
