Music Dashboard

v3

Purpose

Adds an artist detail view toggled via radio buttons and a queue/up-next panel alongside the album grid. Explores multi-view navigation patterns within a music dashboard context.

Prompt

v3: Artist detail view (radio toggle), queue/up-next section.

Add a view toggle (Albums / Artist) using radio buttons. The Artist view shows a hero section with a circular avatar, artist name, stats, and a popular tracks list. Add a queue/up-next panel on the right side of the album grid showing the current track and upcoming songs with thumbnails and durations.
