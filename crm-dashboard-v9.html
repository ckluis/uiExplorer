Purpose

A 10-variant origination series exploring CRM dashboard designs for real estate agents. Each variant takes a radically different approach to organizing contacts, deals, and workflows.

Variant Focus

"Schedule" — Calendar-First CRM. Time is your only real resource. A physical day planner metaphor where the weekly calendar drives all activity. Contacts and deals organized by when things happen, with an unscheduled backlog creating visual pressure.

Prompt

This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

v9: "Schedule" — Calendar-First CRM

Tagline: "Time is your only real resource."
Core metaphor: A physical day planner — time is the primary axis, contacts and deals organized by when things happen.

Layout: Large week-view calendar (~75%, right side) with hour rows (8am-6pm). Each event block is a rich mini-CRM card (contact name, property, deal stage badge). Left panel (~25%, ~300px) with "Unscheduled" items at top and today's agenda below. Color-coded by activity: blue=showings, green=meetings, amber=deadlines, purple=closings.

Key UI elements: Rich event blocks as mini-CRM cards. "Unscheduled" queue creating visual pressure. Day/Week toggle. Deadline markers with countdown badges. Time-block category zones. Availability gaps with dotted borders. End-of-day summary bars. Week header with dates.

Color/mood: Clean, structured. White background with subtle grid. Pastel event blocks. High-saturation red deadline markers. Organized, professional — premium leather planner.

Role toggle: Solo Agent sees personal schedule. Team Lead gets multi-row view with conflict detection. Brokerage Admin sees high-level capacity view with volume bars.
