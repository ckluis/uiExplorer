Agent Columns

v2

Purpose

Evolves the Linear-style orchestration dashboard with richer manager description cards, cross-column dependency visualization via color-coded badges, and parallel/sequential phase grouping within each column. Shows the complexity of agent coordination — what can run simultaneously vs what must wait.

Prompt

I think we can do a better job of showing some of the complexities with dependencies. And each column should have a little more detail (maybe a card) that explains what its doing and what its reponsible for managing before the subagents that are working. We should also show what can work in parrallel vs not.

Think about the state of art and add some of those flourishes to this design (but do not deviate from the clean linear quality design).
