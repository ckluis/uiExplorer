Agent Columns

v1

Purpose

A Linear.com-inspired orchestration dashboard where an orchestrator sits on top and manager columns beneath it show sub-agents as single-line items with progress indicators. The meta-project is generating a UI framework — each manager owns a piece, each sub-agent is a task.

Prompt

Time for a new experiment. We already attempted an human/agent experiment autonomously, but frankly I feel like it failed.

I'd like to consider something more column like. The orchestrator sits on top and there are columns beneath it for managers who each have sub-agents. Each sub-agent is represented with a single description line (with a hover over to see more) and a progress indicator and/or crossed off when done.

The project could be generating an experimental UI framework (just to keep it meta) you could explain what the orchestrator is doing and its managers and their subagents to run experiements. Each manager would be an experiment and each subagent the individual attempts.

We need be REALLY clean and crisp with the ui like linear.com designed this framework. Not atlassian.
