Platform Dashboard

v1

Purpose

Explores a cross-product ERP dashboard with AI-derived insights at the top (with thumbs up/down/bookmark feedback), followed by Apple Store-style horizontal carousels per product line with stat cards, action cards, and todo lists.

Prompt

Read the claude.md first.

We want to create a new experiment for a platform dashboard that shows insights from multiple products (use a fake erp in this case) the insights should be grouped at the top and have very clear value and label which products were used to derive these insights and guide the user to specific actions. We want them to be able to see what to do. Like netflix we want to know if these insights dervied value so besides tracking usage/behaviors in logs we also want a thumbs up/down on recommendations or a "ill come back to this" type option.

Below the insights the dashboard should be broken up into carousels of stat cards like apple.com/store per product line with calls to action and/or links as appropriate.

These carousels should have a dot indicator below them to show which ones are visible on page and a click option to scroll over to other cards (and/or) a left/right arrow below each carousel on the bottom left

It is up to you if you think it makes sense to group the insights into a carousel like this with different sized cards based upon importance. Make sure one of the carousel cards is a todo list for each product line.
