Human/Agent Orchestration

v2 Fully Autonomous

Purpose

Exploring agent orchestration as geological strata — a cross-section of earth where each orchestration layer becomes a distinct geological stratum. Deeper = more specialized, more compressed reasoning. Part of a 10-variant research series.

Variant Focus

Geological Strata — the human prompt sits at the surface as sky/atmosphere. The orchestrator is sedimentary rock (loose, readable). Sub-agents become metamorphic (compressed, denser). Deep sub-sub-agents are igneous (intense, crystallized logic). A "core sample" shows how reasoning transforms with depth.

Include a geyser element representing an AskUserQuestion erupting from depth to surface. Use earth tones, CSS texture patterns, and increasing visual density/compression with depth.
