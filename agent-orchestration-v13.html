Human/Agent Orchestration

v13 Fully Autonomous

Purpose

Enhanced iteration of the Orchestra Score (v3) — adds device preview framework and CSS-only interactions. Part of a 20-variant research series.

Variant Focus

Device preview + checkbox instrument muting, hover note tooltips, time scrubbing via radio buttons, and fermata hover reveals question text.

Prompt

This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

Build agent-orchestration-v13.html as an enhanced iteration of v3 (Orchestra Score). Take the existing Orchestra Score and make two changes: 1. Wrap in device preview window framework (desktop/mobile/fullscreen). 2. Add CSS-only interactions: checkbox per instrument staff to mute/highlight that agent line, :hover on notes shows action detail tooltips, radio buttons for time position (different moments in orchestration), fermata symbols pulse on :hover revealing question text. Mobile: stacked staves (3-4 visible), simplified notation. Preserve the parchment/ink score aesthetic.
