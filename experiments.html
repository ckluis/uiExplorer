The UI Lab

A living collection of interface experiments built at the intersection of human creativity and AI capability — all in pure HTML and CSS, no JavaScript involved. This is the showcase; the framework is the point.

Run your own lab

This whole system is forkable. Clone the repo, run the /uiExperiment commands, and every experiment you create documents its own making. See how →

How It Works

Some experiments are born from collaboration — a human and an agent trading ideas, each prompt refining the last. Others are fully autonomous: an agent given a direction and left to run, iterating on its own without human intervention.

On Documentation

Every experiment tries to document its own making — the prompts, the decisions, the creative back-and-forth. For collaborative sessions, this provenance is precise. For autonomous runs, the documentation captures the spirit of what was explored, even when the exact prompt trail is more summary than transcript.

What You're Looking At

Think of it as a creative lab notebook: part portfolio, part process archive, part proof-of-concept for what happens when humans and agents experiment together — or when agents experiment alone.
