This version was generated autonomously by a sub-agent. The prompt below is the exact prompt given to the build agent — it was crafted by the orchestrating agent based on the origination prompt and research phase.

v3: "Stream" — Timeline/Activity Feed

Tagline: "Everything that happened, as it happened."
Core metaphor: A social media news feed — a chronological record of every interaction across all contacts and deals.

Layout: Single dominant column (centered, ~700px) with activity cards flowing vertically. Left sidebar (~240px) with filters by contact, activity type, date range, deal stage. Right rail (~200px) with "Today's Summary" stats and upcoming items. Date separator lines between event clusters.

Key UI elements: Activity cards with left-side icon rail (phone, envelope, house, dollar) connected by vertical timeline line. Sticky date separators. Filter chip bar. Expandable detail on cards. Contact avatar clusters. "Quiet period" warning bands. Milestone markers. Daily digest summary card.

Color/mood: Light, airy, generous whitespace. Pale background with white cards. Activity icons in muted colors (blue=communication, green=showings, amber=offers, purple=closings). Calm, readable, archival.

Role toggle: Solo Agent shows personal activities. Team Lead interleaves team activities with agent badges. Brokerage Admin defaults to highlights only with density toggle and brokerage KPIs.
